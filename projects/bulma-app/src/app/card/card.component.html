Basic Card Details

The most basic card needs only an <bu-card> element with some content. However, Bulma components provides a number of preset sections that you can use inside of an </bu-card>
Element Description
<bu-card-header> Card Header
<bu-card-title> Card Title
<bu-card-content> Primary card content. Intended for blocks of text
<bu-card-footer> Section anchored to the bottom of the card
